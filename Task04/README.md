# GCD Game (Наибольший общий делитель)

Веб-приложение на **Laravel** для игры «НОД».

Проект выполнен в рамках дисциплины  
**«Технологии разработки серверных приложений на PHP»**.

## Описание

Пользователю показываются два случайных целых числа. Нужно вычислить и ввести **наибольший общий делитель** (НОД).  
Результат каждого раунда сохраняется в **SQLite** и доступен на странице истории.

Приложение реализовано на **Laravel**:
- маршрутизация через `routes/web.php`
- контроллеры для главной страницы, игры и истории
- шаблоны на **Blade**
- хранение данных через **Eloquent ORM**
- имя игрока сохраняется в сессии

## Функциональность

- **Главная страница**
  - ввод имени игрока
  - переход к игре
- **Игра**
  - генерация двух случайных чисел
  - ввод ответа
  - вывод результата: верно/неверно
  - сохранение раунда в базу данных
- **История**
  - просмотр всех сыгранных раундов
  - отображение времени, игрока, чисел, ответа и результата
- **Сброс имени**
  - очистка текущего игрока из сессии

## Требования

- PHP 8.1+
- Composer

- Laravel

## Установка

Из корня репозитория перейти в каталог `Task04`:

```bash
cd Task04
```

Установить зависимости:

```bash
composer install
```

Создать файл окружения:

```bash
cp .env.example .env
```

Создать ключ приложения:

```bash
php artisan key:generate
```

Создать SQLite-базу данных:

```bash
mkdir -p database
touch database/database.sqlite
```

Настроить подключение к SQLite в файле `.env`:

```env
DB_CONNECTION=sqlite
DB_DATABASE=database.sqlite
```

В файле `config/database.php` для секции `sqlite` используется путь через `database_path(...)`, чтобы проект корректно запускался на любом компьютере.

Выполнить миграции:

```bash
php artisan migrate
```

## Установка через Makefile

Если в системе доступна утилита `make`, можно выполнить все основные шаги одной командой:

```bash
make install
```

## Запуск

Запустить встроенный сервер Laravel:

```bash
php artisan serve
```

После запуска приложение будет доступно по адресу:

- http://localhost:8000/

## База данных

SQLite-файл хранится в каталоге:

`Task04/database/database.sqlite`

Это соответствует требованиям задания: файл базы данных располагается внутри каталога `database`.
