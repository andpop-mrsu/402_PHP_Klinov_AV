# GCD Game (Наибольший общий делитель)

Игра на PHP: по двум случайным целым числам нужно вычислить **НОД** и ввести его в терминале.

Проект выполнен в рамках дисциплины  
**«Технологии разработки серверных приложений на PHP»**.

## Описание

Пользователю показываются два случайных целых числа. Нужно вычислить и ввести **наибольший общий делитель** (НОД).  
Результаты раундов сохраняются в **SQLite** и доступны для просмотра на странице истории.

Архитектура выполнена в стиле **Page Controller**: каждая страница — отдельный PHP-контроллер в `public/`.

## Функциональность

- **Главная страница**:
  - ввод имени игрока (сохраняется в сессии)
  - ссылки на игру и историю
- **Игра**:
  - показывает 2 случайных числа
  - принимает ответ и выводит результат (верно/неверно + правильный ответ)
  - сохраняет запись в SQLite: время, имя, числа, ответ, корректность
- **История**:
  - таблица: время | игрок | числа | ответ | верно/неверно

## Требования

- PHP 8.0+ (желательно 8.1+)
- PDO + SQLite (расширение `pdo_sqlite`)
- Любой браузер

## Установка и запуск

Перейти в каталог Task02 и запустить встроенный сервер PHP:

```bash
php -S localhost:3000 -t public
```

Открыть в браузере:

- http://localhost:3000/


## Хранилище данных

SQLite база создаётся автоматически по пути:

`Task02/db/gcd.sqlite`
